#Analista de Partidos de Fútbol

Aplicación web sencilla desarrollada con **Flask** que permite registrar y visualizar partidos de fútbol, además de generar estadísticas básicas y probabilidades de resultado (local, empate, visita).

El proyecto tiene un diseño moderno con temática futbolera y está pensado con fines **académicos y demostrativos**.

---

##Funcionalidades (Futuras)

- Visualización de partidos registrados
- Cálculo automático de:
  - Total de partidos
  - Victorias locales
  - Empates
  - Victorias visitantes
- Probabilidades estimadas de resultado
- Interfaz moderna con estilo futbolero
- Navegación simple entre vistas

---

## Tecnologías utilizadas

- **Cursor**
- **Python 3**
- **Flask**
- **HTML5**
- **CSS**
